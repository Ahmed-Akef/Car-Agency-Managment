# Car Agency Management System

## Description
This project is a Car Agency Management System developed using ASP.NET.
It manages cars, customers, and reservations.

## Project Database
The database files are located in the folder:
Project Database

This folder contains the SQL file used to create tables and manage data.

## How to Run
1. Open the project using Visual Studio
2. Configure the database connection
3. Run the SQL file to create the database
4. Run the project
